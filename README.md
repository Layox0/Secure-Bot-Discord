✅ FONCTIONNALITE :

Anti-spam automatique

Action d'accès à l'écriture dans le salon
+lock 
+unlock

Action d'accès au chanel
+fermer
+ouvert

Action de TimeOut
+mute
+unmute

Action de Mdération
+bannir
+kick

🔧 CONFIGURATION                                                                                                      

1. Installe les dépendances :
   pip install discord.py==1.7.3

2. Remplace "YOUR_DISCORD_BOT_TOKEN" par ton token dans config.py

3. Remplace OWNER_ID par ton ID Discord

4. Lance le bot avec :
   python bot.py

5. toute les commandes sont accessibles uniquement au OWNER_ID
